# BanJona

A lightweight banjo-themed web landing page built for the BanJona repository.

## What’s included

- `index.html` — simple static homepage with banjo design and chord interaction
- `styles.css` — responsive styling with a dark accent theme
- `script.js` — random chord generator and interaction support

## Run locally

Open `index.html` in your browser or use a local server for best results.

## Development

This repository is currently scaffolded as a simple static site. You can extend it with:

- a banjo learning section
- chord diagrams and tablature
- audio samples and jam tracks
- user-contributed song library
